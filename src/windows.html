Windows

Go and get Windows 10

Windows 10 ist der Name des aktuellen Betriebssystems von Microsoft. Somit ist es der direkte Nachfolger von Windows 8.1. Windows 10 wurde offiziell Ende September 2014 enthüllt, dabei gab es von Microsoft unter anderem einen ersten Ausblick auf die Rückkehr des Startmenüs, die Möglichkeit zum Wechsel zwischen mehreren Desktops sowie frei veränderbare Fenstergrößen bei mo­der­nen Apps. Der Code­name lautete "Threshold". Die finale Version ist im Sep­tem­ber 2015 erscheinen. Zu den Besonderheiten von Windows 10 gehören ein stark überarbeitetes Start­me­nü, der Sprachassistent Cortana, der Microsoft Edge-Webbrowser und die direkte Einbindung des Microsoft Store, über den Nutzer Apps beziehen können. Microsoft gibt an, künftig keine komplett neuen Versionen des Betriebssystems mehr zu veröffentlichen. Stattdessen wird Windows 10 seit seiner Veröffentlichung fortlaufend mit Updates versorgt, welche oft auch neue Funktionen mitbringen. Aktuell ist das Windows 10 Mai 2021-Update mit der Versionsnummer 21H1 verfügbar.
